Erfasst Aufrufe über den URL-Parameter "campaign" und ordnet sie Kundenkampagnen zu.

== Description ==
Dieses Plugin ermöglicht dir:
- Kampagnen für Kunden im Backend anzulegen
- automatisch Tracking-Links zu erzeugen
- Klicks inkl. Zeitpunkt, URI und Referer zu speichern
- Statistiken in einem Dashboard anzuzeigen
- Dashboard nach Kunde/Zeitraum zu filtern inkl. Pagination und Eintragsanzahl

Wichtig:
- Ohne campaign-Parameter wird nichts gespeichert.
- Bei unbekanntem campaign-Code wird nichts gespeichert.

== Installation ==
1. ZIP in WordPress unter Plugins > Installieren > Plugin hochladen importieren.
2. Plugin aktivieren.
3. Menü "Kunden Tracker" öffnen.
